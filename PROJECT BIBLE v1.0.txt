PROJECT BIBLE v1.1
项目名称：枪声之前：成都城在一天之内变了样
项目类型：《中国近现代史纲要》10分钟AI历史短片
创作方式：纪录片旁白 + 电影化重演
版本：v1.1

一、题材定义
本项目聚焦 1911 年保路运动中的“成都血案前后24小时”。

注意：
1. 不是拍整个保路运动
2. 不是拍完整辛亥革命
3. 不是做课件式历史复述
4. 而是通过一个更小、更可控、更具电影感的切口，表现一座城市如何在一天内从异样、停摆、汇聚、枪响，走向奔逃、救治、组织、传信与反抗外溢

二、主题句
枪声响起之前，一座城已经先变了样。
赵尔丰的枪声没有恢复秩序，反而推动了更大范围的反抗与历史转折。

三、历史表达目标
本片要传达的不是“成都当场胜利”，而是：
1. 铁路国有与路权风波伤害了四川广泛人群利益
2. 成都血案把本已紧张的社会矛盾推向爆裂
3. 血案之后，人民并未只是被动逃散，而是迅速出现自救、互助、传信、重新组织和反抗外溢
4. 这种反抗不靠大规模大战来表现，而通过街巷、药铺、路口、江边、城门等局部空间来呈现
5. 镇压没有压回沉默，反而推动更大范围的社会动员和历史连锁反应

四、叙事视角
采用小人物穿城视角。
主视角为一名18岁左右的成都青年学徒，暂定名“阿生”。

他的功能不是改变历史，而是：
- 看见
- 跑动
- 传递
- 躲避
- 帮忙
- 被卷入
- 从懵懂走向清醒

五、人物设定
1. 主角：阿生
- 年龄：18左右
- 身份：商会/报馆/药铺体系中的跑腿学徒
- 外形：偏瘦、灰褐旧布短褂、布鞋、旧毛巾
- 初始状态：机灵、懂看人脸色、但不真正理解大局
- 后期状态：不再只是害怕，而是变得沉默、冷静、知道事情不会回到原样

2. 反复出现的配角类型
- 青年学生：激进传播、抄写、喊话、传单、组织
- 商会/掌柜/股民：旧秩序受损、利益焦虑、隐忍愤怒
- 茶馆老板：平民避险本能、街面温度变化
- 清兵/巡防营：暴力秩序、机械压迫
- 药铺人物/抬伤员者：事件的肉身代价与民间自救
- 传信者/江边放木板者：反抗与扩散的象征

六、固定场景池
仅围绕以下空间展开，尽量不无限扩景：
1. 商会/议事厅/铁路公司室内
2. 茶馆外街/成都街市
3. 督署前广场
4. 窄巷/后门/通道
5. 锦江边/桥头/河岸
6. 药铺/临时救护点
7. 城门/城墙远景
8. 空街/屋顶/旗帜/城市空镜

八、镜头语言
优先使用：

- 越肩

- 从人群缝隙中看

九、叙事结构
全片建议按 7 段推进：
1. 异样
2. 停摆
3. 汇聚
4. 枪响
5. 奔逃与救治
6. 组织与传信
7. 次日与外溢

十、核心叙事原则
必须同时满足：
1. 先写“城里空气变了”
2. 再写“人群汇聚与对峙”
3. 枪响之后必须有痛感，但不能只停留在受难
4. 必须体现自救、互助、组织、传信与外溢
5. 不把人民只写成被动受害者
6. 不把保路运动写成简单单场胜利
7. 结尾落在：秩序并未恢复，反而更大风暴即将到来

十一、适合表现“反抗”的方式
优先表现：
- 药铺中的分工与救护
- 巷口封门与简易路障
- 递送口信
- 木板传信、水路传信
- 人群表情从惊惶转向决绝
- 城外路口、县份响应的前夜意象
- 刀枪火把的准备感
- 从“请愿”到“不会再退”的空气转变

不优先表现：
- 大规模万人混战
- 长时间正面战争
- 夸张英雄冲锋
- 当场凯旋式胜利

十二、生产原则
1. 先确定史实边界与剧情结构
2. 再拆分镜
3. 再做角色定妆与场景定板
4. 再出关键首帧/尾帧
5. 再转视频镜头
6. 最后做剪映拼接、音效、字幕与节奏控制

十三、一致性规则
1. 阿生必须有固定脸型、年龄感、服装逻辑
2. 督署、街巷、茶馆、药铺、锦江要像同一座城
3. 所有镜头必须服务同一套气质，不得忽冷忽热、忽写实忽奇幻
4. 所有新建议都必须优先检查是否与本 Bible 冲突
5. 历史细节必须先搜索确认；若无法确认，只能做“创作性合理补足”，并明确标注

- 单纯热血胜利叙事
