# 史纲AI视频

## 分工

zyc和grl负责前端内容，包括史实整理、情节结构、分镜、图片生成、修图补图、角色定妆和场景定板；zsh负责后端落地，包括视频提示词、视频生成、镜头续接替换、剪映拼接，以及背景音/环境音/音效处理。

我们统一使用gemini里的gem，整套提示词在`system.txt`，包含主 Gem 指令、PROJECT BIBLE （上传到参考资料里的）和几个任务用的初始提示文案。大家不用全都用一遍，只需要按自己负责的部分取用，但都要统一遵守同一套主指令和 PROJECT BIBLE，这样才能保证剧情、人物、场景、色调和镜头语言始终一致，不会各做各的。

## 项目结构

```
ShiGangAIVideo/
├── history.txt          # 生成历史剧情的提示词
├── PROJECT BIBLE v1.0.txt # 项目圣经，上传到 gen 的参考资料里，包含人物、场景、色调、镜头语言等统一规范
├── REAMD.md             # 本文件
├── storyboard.txt         # 生成分镜的提示词
├── system.txt           # 系统提示词
├── video.txt            # 生成视频的提示词
├── figures/             # 存放图片、角色定妆和场景定板
```
